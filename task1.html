PROJECT


1. Plant disease prediction using deep learning :
      In this project we showed how neural networks can be used for plant infection detection with the help of image classification. In this project first we add th photo of the leaf of a plant and then it will show whether the plant has any sort of disease or it is a fresh leaf without any disease .Three different architectures were used including ResNet152V2 , InceptionV3, Resnet50. It was observed that ResNet152V2 gave us the best outcome on the test set with the accuracy of 0.98 and precision of 0.94.
2. PC Control using hand gestures :
      We aim to reduce use of hardware of the computer like mouse and keyboard and to create a service whic is wireless so we can operate multiple system without changing our stationary position. We used arduino UNO and ultrasonic sensors in the hardware whereas in the software arduiono IDE and Python was used .
